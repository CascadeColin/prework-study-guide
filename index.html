Enter for Git notes here

  • Enter your Git notes here
  • git status: checks what branch i am on
  • git checkout -b branch-name: creates a new branch (replace branch-name with desired name)
  • Git allows me to use Unix commands to communicate between the IDE (local computer) and GitHub
  • This practice is critical for version control
  • Proper use allows me to revert to previous changes so I don't have to restart from scratch (like the first time I did it)
  • Saving often gives more "checkpoints" so I lose less data and time when I need to revert
  • Using a tilde (~) before calling a file directory is a wild card to access the folder
  • example: "cd ~/bootcamp" will call the directory "c:/users/colin/bootcamp"
